TASKFLOW - PHP & MYSQL TASK APP

HOW TO RUN:

1. Install XAMPP
2. Start Apache and MySQL
3. Move folder into:
   C:\xampp\htdocs\
4. Open phpMyAdmin:
   http://localhost/phpmyadmin
5. Import database.sql
6. Open:
   http://localhost/Taskflow

FEATURES:
- Modern interactive UI
- Add Tasks
- Delete Tasks
- Responsive design
